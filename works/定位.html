4.绝对定位absolute

先看默认显示，B包含3个DIV，B1,B2,B3从上往下排列。 
<html>

<head>
    <title>position</title>
    <style type="text/css">
        div {
            border: 1px solid #999999;
            margin: 1px;
            height: 50px;
        }

        .AA {
            position: fixed;
            bottom: 0;
            right: 0;
            width: 100px;
            background: RED;
            color: white;
        }

        .B {
            height: 200px;
            position: relative;
        }

        .B div {
            width: 200px;
            height: 50px;
            border: 1px solid red;
        }
    </style>
</head>

<body>
    <div class="A">
        A
    </div>
    <div class="AA"">
        AA
    </div>
    <div class=" B">
        <div class="b1">B1</div>
        <div class="b2">B2</div>
        <div class="b3">B3</div>
    </div>
    <div class="C">
        C
    </div>
    <div class="D">
        D
    </div>
    <div class="E">
        E
    </div>
</body>

</html>
            
1.首先将B元素position设为relative；
2.将B1,B2 position设为absolute;

B1,B2绝对定位后，脱离文档流，B3顶上。
B1,B2的位置，基于最近的已定位祖先元素，本例中就是第一步设置了position为relative的B元素。
B也可设置为fixed,absolute,B1,B2的位置相对于B效果是一样的。但B本身的位置就会变化，所以最常用的是将父元素B设为relative，B元素位置本身没有任何变化。

top:100px; 则上面距离B顶部距离为100.
right:20px;则右边距离B右边框距离20px;
left:20px;左边距离B左边框距离20px;
bottom:20px;下边距离B下边框距离20px;
        
<html>

5.相对定位

1.相对定位最简单最容易理解，即相对于他在文档流中原本的位置，进行一个偏移。可用于微调。
2.从示例可看出，后面的C\D\E不受B元素位置的变动影响。
        
<html>

<head>
    <title>position</title>
    <style type="text/css">
        div {
            border: 1px solid #999999;
            margin: 1px;
            height: 50px;
        }

        .AA {
            position: fixed;
            bottom: 0;
            right: 0;
            width: 100px;
            background: RED;
            color: white;
        }

        .B {
            position: relative;
            left: 20px;
            top: 20px;
        }
    </style>
</head>

<body>
    <div class="A">
        A
    </div>
    <div class="AA"">
        AA
    </div>
    <div class=" B">
        B
    </div>
    <div class="C">
        C
    </div>
    <div class="D">
        D
    </div>
    <div class="E">
        E
    </div>
</body>

</html>
