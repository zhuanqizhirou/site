4.元素选择器

<html>

<head>
    <title>css学习</title>
    <style type="text/css">
        div {
            color: red;
        }
    </style>
</head>

<body>
    <div>
        只有一个最基本的div元素，其他属性id、class等等都没有。
    </div>
</body>

</html>
            
1.元素【即html元素】选择器是最基本的选择器！
因为最基本的网页，不需要id，不需要class也能通过元素设置样式。
2.w3school中有专门几节讲选择器，从元素选择器开始。
2.上面的派生选择器，也是基于元素选择器。【div span】,选择div元素下的span元素，可以看作是2个最基本的元素选择器组合起来。

5.更复杂的派生选择器

1.派生选择器是css2中的名称，在css1中叫做“上下文选择器”。
2.在上面简单派生选择器示例中，是由2个最基本的元素选择器组成
3.派生选择器还可由其他选择器通过空格连接组成。如：
2个类选择器组合 .a .b{color:red;}  
1个类选择器与1个元素选择器组合 .x div{color:blue;}
        
<html>

<head>
    <title>css学习</title>
    <style type="text/css">
        .a .b {
            color: red;
        }

        .x div {
            color: blue;
        }
    </style>
</head>

<body>
    <div class="a">
        <div class="b">.a .b</div>
        <div class="c">.a .c</div>
    </div>
    <div class="x">
        <div>.x div</div>
        <span>.x span</span>
    </div>
</body>

</html>

6.从简单到负责的选择器

1.html组织页面内容，css优化内容的显示
2.一个纯的html，可以通过元素选择器设置特定元素的样式
3.如果想为页面某一个指定的内容设置样式，可以通过id选择器。
4.如果想为页面多个内容设置相同样式，可以通过为选定内容的元素上增加class，通过类选择器设置样式。
5.其他更复杂、精准的，使用派生选择器，组合起来达到目的。
